# Exp 2.2 - Bootstrap Cards

A React application showcasing Bootstrap card components with various layouts and styles.

## Features

- Movie genre cards with images
- Hover effects and animations
- Professional card layouts
- Responsive grid system
- Clean white background design

## Technologies Used

- React 18
- Bootstrap 5
- Vite

## Installation

```bash
npm install
npm run dev
```

## Components Demonstrated

- Bootstrap cards with images
- Card headers and bodies
- Hover transformations
- Grid layout system
- Professional styling with shadows
