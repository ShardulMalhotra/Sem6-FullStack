# Exp 2.1 - Bootstrap Buttons

A React application demonstrating various Bootstrap button components and interactions.

## Features

- Interactive button showcase with click counters
- Multiple button variants (Primary, Secondary, Success, etc.)
- Button groups and loading states
- Professional white background design
- Responsive layout

## Technologies Used

- React 18
- Bootstrap 5
- Vite

## Installation

```bash
npm install
npm run dev
```

## Components Demonstrated

- Bootstrap buttons with different variants
- Button groups
- Interactive click counters
- Loading button states
- Hover effects and transitions
